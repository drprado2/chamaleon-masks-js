Chameleon Masks
JS

Uma biblioteca JS com sintaxe simples e declarativa para criação de mascaras customizáveis

GET STARTED GITHUB
Apenas JS

A biblíoteca foi desenvolvida apenas com ECMA Script, ou seja ela é livre de Jquery, ela possui Babel configurado para gerar um código compatível com a maioria dos browsers para se encaixar em qualquer projeto.

Completamente Customizável

Chameleon Masks JS foi desenvolvida para atender a qualquer mascara de campos numéricos, com uma simples sintaxe declarativa você cria qualquer mascara.

Código Aberto

Nossa biblioteca possui código aberto e está disponível no github, você pode analisar o código, criar um fork e fazer suas próprias alterações ou contribuir com o projeto tornando ele cada vez mais completo.

Mascaras Comuns

Com o uso de mascaras comuns você consegue ter mascara para qualquer campo númerico, como documentos (CPF, CNPJ), telefones, CEPs.

assignment_ind
account_balance
local_phone

Multi Mascaras

A multi mascara é uma feature muito interessante, onde é possível colocarmos várias mascaras em um mesmo input as mascaras serão apresentadas da menor para a maior, quando o input atinger o comprimento da próxima mascara ela será selecionada automaticamente.

assignment_ind
local_phone 
                
    Chameleon.init(cpf, "999.999.999-99");
    Chameleon.init(cnpj, "99.999.999/9999-99");
    Chameleon.init(cep, "99999-999");
    Chameleon.init(telephone, "(99) 9999-9999");

Mascaras de Dinheiro

Através da união de 3 features (infinit, reverse, money) podemos ter mascara uma mascara de dinheiro rica e completa, onde os valores são adicionados de forma reversa (centavos, centenas, milhares...) e podemos acrescentar o cifrão da moeda desejada.

monetization_on
monetization_on

Limitar Valores Mascaras

Com a feature de limitar valores podemos por exemplo em um input de data não aceitar nos dias valores menores que 1 e maiores que 31, e para limitar o valor aceito é extremamente simples basta acrescentarmos [inicio-fim] ao lado do bloco da mascara, Chameleon não permitira que o usuário insira valores fora desse límite, e ainda chamará uma função de callback quando isso ocorrer.

date_range
watch_later

Mascara Infinita

Com a mascara infinita podemos montar uma mascara que aceitará valores infinitos, porém cada vez que fechar um bloco ela colocara nossa mascara, note que usamos essa opção com a mascara de dinheiro já que aceita infinitos valores.

phone
all_inclusive
