Chameleon Masks
JS

A complete and friendly JavaScript library to solve input masks problems

GET STARTED GITHUB
Apenas JS

A biblíoteca foi desenvolvida apenas com ECMA Script, ou seja ela é livre de Jquery, ela possui Babel configurado para gerar um código compatível com a maioria dos browsers para se encaixar em qualquer projeto.

Completamente Customizável

Chameleon Masks JS foi desenvolvido para atender a qualquer mascara de campos numéricos, com uma simples sintaxe declarativa você customiza sua própria mascara, por exemplo "999.999-99" seria uma mascara válida.

Código Aberto

Nossa biblioteca possui código aberto e está disponível no github, você pode analisar o código, criar um fork e fazer suas próprias alterações ou contribuir com o projeto tornando ele cada vez mais rico.

Mascaras Comuns

Com o uso de mascaras comuns você consegue ter mascara para qualquer campo númerico, como documentos (CPF, CNPJ), telefones, CEPs
