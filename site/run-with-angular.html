Run With Angular

Veja como é simples colocar máscaras nos seus inputs em Angular.

Code

1 - Fazer a instalação do pacote pelo Npm

npm install chameleon-mask-js

2 - Importar no seu componente.ts a classe Chameleon

import Chameleon from 'chameleon-mask-js';

3 - Criar o input no template html, de tal maneira que seja possível selecionar ele com um seletor CSS.

<input type="text" name="phone" id="phone" [(ngModel)]="myVarPhone" />

4 - Na função ngOnInit() do seu componente ligar a mascara.

ngOnInit() {
  Chameleon.init('#phone', '(99) 9999-9999', '(99) 99999-9999');
}
